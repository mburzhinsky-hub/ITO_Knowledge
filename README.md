# База знаний ИТО — MVP 1.0

Автономный многостраничный сайт без внешних библиотек и сервера.

## Запуск

Откройте `index.html` в браузере. Для более стабильной работы локального хранилища и выгрузок можно запустить папку через локальный сервер:

```bash
python3 -m http.server 8080
```

Затем открыть `http://localhost:8080`.

## Основная механика

- перетаскивание шаблонов задач из библиотеки в проект;
- автоматический расчёт рабочих дат по зависимостям;
- несколько предшественников у задачи;
- изменение длительности, ролей, прогресса и лагов;
- перемещение полосы Ганта и растягивание её правого края;
- готовые составы полного проекта, СКС, проектирования и ПНР;
- сохранение проекта в `localStorage` браузера;
- экспорт в CSV и JSON;
- печать Ганта.

## Ограничения MVP

- нет общей серверной базы и совместной работы;
- нет импорта обратно в Bitrix24;
- календарь учитывает субботу и воскресенье, но пока не учитывает государственные праздники;
- трудоёмкость основана на ориентировочных длительностях, выведенных из анализа задач, а не на утверждённых нормативах;
- полномочия руководителя ИТО и ведущего инженера помечены как предварительные, поскольку отдельные должностные инструкции не были предоставлены.
