# База знаний ИТО — MVP 1.1

Автономный многостраничный MVP базы знаний инженерно-технического отдела.

## Что внутри

- `index.html` — главная страница и поиск по знаниям;
- `knowledge.html` — каталог инструкций;
- `article.html?id=sks-guide` — полноформатная статья базы знаний;
- `templates.html` — 31 шаблон исполнимых технических задач;
- `standards.html` — стандарты и чек-листы;
- `roles.html` — роли и правила принятия результатов;
- `planner.html` — отдельный инструмент сборки технического плана и Ганта.

## Логика продукта

База знаний является главным слоем системы. Пользователь сначала находит инструкцию, изучает входы, порядок работы, результаты, критерии приёмки и типовые ошибки, а затем при необходимости добавляет связанные исполнимые задачи в план проекта.

Планировщик и Гант являются инструментом базы знаний, а не её заменой.

## Запуск

Откройте `index.html` напрямую или запустите локальный сервер:

```bash
python3 -m http.server 8080
```

Затем откройте `http://localhost:8080`.

## Ограничения MVP

- Данные и сохранённый план хранятся локально в браузере.
- Нет серверной авторизации и совместного редактирования.
- Нормативные длительности являются стартовой оценкой и требуют калибровки на завершённых проектах.
- Праздничные дни не исключаются из расчёта.
